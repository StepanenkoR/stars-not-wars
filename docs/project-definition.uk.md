# Stars Not Wars

## Головне питання
Як людство може перенаправити ресурси з воєнно-руйнівної логіки на розвиток цивілізації, орієнтованої на життя?

## Головна теза
Людство спалює занадто велику частину свого майбутнього на війну і занадто малу — на життя, науку, енергію, здоров’я та довгий цивілізаційний розвиток.

## Мета версії 0.1
Створити мінімальний дата-фундамент і використати його для постановки перших дослідницьких питань.
